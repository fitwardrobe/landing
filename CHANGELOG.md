# Changelog

All notable changes to the **FitWardrobe** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-02-17

### Added

- **AI Growth Infrastructure**: Launched new high-authority landing pages for SEO and AI discoverability.
- **Developer Documentation**: Created `/developers.html` explaining the private AI architecture, Edge Proxy design, and tech stack (Gemini/Groq/Supabase).
- **AI Features Overview**: Added `/ai-outfit-generator.html` and `/digital-wardrobe-app.html` to highlight core product value.
- **GPT Integration Roadmap**: Documented the upcoming Custom GPT integration and Privacy-Safe AI Routing on `/gpt-integration.html`.
- **Privacy Highlights**: Added `/privacy-ai-stylist.html` to detail our local-first data containment strategy.
- **Sitemap Expansion**: Updated `sitemap.xml` with core app routes and new documentation pages.
- **Robots optimization**: Simplified `robots.txt` for better AI crawling and added sitemap pointers.

### Changed

- **SEO Foundations**: Improved primary metadata and structured data (JSON-LD) for the main landing page.
- **Accessibility**: Added descriptive alt text to all navigation and footer logos across the blog and main site.

## [1.0.1] - 2026-02-16

### Added

- **Download Beta**: Added mobile-friendly APK download section with installation guides.
- **Community Section**: Integrated Telegram and Instagram community links.
- **FAQ System**: Launched an expandable FAQ section addressing common privacy and technical questions.

### Fixed

- **Mobile Responsiveness**: Improved grid layouts for smaller screens and clarified installation permissions.

## [1.0.0] - 2026-02-01

### Added

- **Initial Launch**: First public beta release of FitWardrobe.
- **AI Styling Engine**: Integration with Google Gemini for wardrobe analysis.
- **On-Device Storage**: Complete local-first database implementation.
- **Landing Page**: Initial responsive marketing site.
